# Sistema de Recuperación de Información

##Requisitos:
Python
Node.js + npm
Angular CLI (npm install -g @angular/cli)

##Cómo ejecutar el backend:
Entra a la carpeta back/
Instalar las dependencias pip install -r requirements.txt
Ejecutar el backend python back.py
De manera general, El backend se levanta en http://localhost:3000

##Cómo ejecutar el front:
Entra a la carpeta project/
npm install
ng serve
Generalmente, desde el navegador se despliega en el http://localhost:4200
